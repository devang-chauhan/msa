Setting up a Page Layout in Excel is different than the AutoCAD.
I have taken a Project DCI for an example here.
When you open any excel data sheet, you'll find the default screen as shown below:


Here is a stepwise introduction about how to setup a Page Layout:

Step 01: Page Layout

  1. Click on the Page Layout icon in the middle, On Bottom Right of the Excel sheet.
  2. Go to Page Layout tab
  3. Click on Page Setup icon. It will open a Page Setup toolbox.


Step 02: Page Setup

As you can see that some information is leaking out to the another page, which has to be in the same page.


To set it up, Follow the Step 01,
Once the Page Setup toolbox is opened, Do as the following:

  1. Go to the Page tab
  2. Set the Page Orientation that you want
  3. Scaling: If possible, keep the % in Adjust to option to 100%, If information leaks, then only decrease the %.
  4. Select the Paper Size that you want
  5. Keep the Paper quality 600 dpi


Note that the % in Scaling option is 85% but still the information leaks, So I'll have to decrease it so that I get all the information in single page.
Here is the page after decreasing the % of scale:


Recheck if the text is visible in PDF/Print after scaling them down.

Step 03: Margins

Follow Step 01 and open Page Setup.

  1. Go to Margin
  2. Change all the Margins to 0.5 (Everyone has to follow this standard in every excel files)
  3. Center on page: Horizontally
  4. Hit OK


Step 04: Sheet Setup

Follow Step 01 and open Page Setup.

  1. Go to Sheet
  2. Print Area: Select all the cells that you want to include in print area. (Only apply this when file has information that you don't want to print)
  3. Rows to repeat at top: Select the row that you want to repeat in every cells:
        For an example, I want the row to appear in my every page which is highlighted in image below, 
        Same theory applies for Columns to repeat at left.
     4. Hit OK.


Step 05: Plot

Only Plot the sheet when you have applied all the Steps listed above.
Plotting the sheet in excel is almost same as to plot it in AutoCAD.

  1. Press Ctrl + P
  2. Name: Select the printer that you want to print from
  3. Click on Properties
  4. Make sure if Factory Defaults is selected
  5. Select the paper size you want
  6. Select the paper source. It maybe varied in every printer. (Eg. My printer holds A4 size papers in Tray 1, So the Tray 1 is selected)
  7. Hit OK
  8. Pages: Enter the numbers of your Excel pages that you want to print
  9. Number of copies: Enter the number of copies that you want
  10. Hit OK in Plot toolbox to give a Print

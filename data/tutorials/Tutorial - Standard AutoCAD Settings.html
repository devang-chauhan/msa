1. Object Snap

Here is the tutorial for you to use object snapping right way,
Which will make it easy for you to draft and save your time.

To open Object snap settings in AutoCAD file, Enter 'SE'
A toolbox called 'Drafting Settings' will be opened.


You'll see different Options in Object Snap modes.
The title defines themselves and no definition will be needed to define them.

Selecting only options which are used frequently will save your time and here's the list of them,
And keeping these listed Object snap modes Active will be helpful for us to maintain the Standards.

Endpoint
Midpoint
Quadrant
Intersection
Insertion
Perpendicular
Apparent Intersection

Keep the rest Object snap modes Inactive.

You'll need other snapping modes too surely, but keeping them active all the time will consume time only.
Whenever you need any of Inactive snap modes, You can enter the name of it in worksheet.
Here's How it works:



You can activate other object snap modes temporarily another way, by pressing Shift + Mouse 'Right' button.



2. 3D Object Snap

If you're facing any problem tracking a 3D Object snap, and find out that your drawing is being drafted in Z axis also,
Then here's how you can stop tracking 3D Object Points.

  1. Enter command SE and go to 3D Object Snap
  2. Clear All, Apply and exit.


If you find out that your drawing was drafted in 3D, Separate tutorial explaining How to convert 3D drawing into 2D will be created.


3. Color Settings

The background color of the Model space and Layouts should be Black for every user.
Here's the tutorial about how to change the background color:

Step 01: Display Options

  1. Enter 'OP' command, which will open 'Options' Toolbox.
  2. Go to Display.
  3. Click 'Colors'


Step 02: Changing the color

After clicking 'Colors', Another toolbox called 'Drawing Window Colors' will be opened.
Now,

  1. Uniform background color of 2D model space should be Black
  2. Uniform background color of Sheet / layout should be Black
  3. Apply & Close



3. Save Settings

Make sure that it is being saved in Autocad 2004/ LT2004 Drawing (*.dwg) type.
If not, Save your file as Autocad 2004/ LT2004 Drawing (*.dwg) type.


Everyone in the firm has to follow these standards.

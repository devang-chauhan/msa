Tutorial - Reducing file size in AutoCAD

Created:14-05-2016 14:58
Updated:16-05-2016 18:52
Author:Bijit Soni


DGNPURGE is a particular command that is used to compress the size of the AutoCAD file.
[LT users will not be getting a benefit of this command as LT versions don't support it.]

Note that this is not an inbuilt command and it has to be loaded externally, by using NETLOAD command in autocad.

Here is a stepwise introduction to load the file 'DgnLsPurge.dll' -which is the source file of the command that you're going to use.
 
Step 01: Open File

Open your AutoCAD drawing that you want to compress the size of.
Here I have attached the image of the file that I want to compress, showing it's original file size for an example.

 

Step 02: Load DGNPurge      

Load 'DgnLsPurge.dll' by using NETLOAD command in AutoCAD drawing.

                    

Step 03: Select DGNPurge    
 
Find 'DgnLsPurge.dll' from the folder;
   
Now, You'll find all the AutoCAD installation related files in this folder. Possibly, finding particular file might be tough.
So do as described in an Image,

  • Option 1: Enter the name of the file (DgnLsPurge.dll) in the 'File name' box and Open it.


  • Option 2: Find the file manually.

               

Now you're done with loading the file.

Step 04: Using DGNPurge    

  1. DGNPURGE (Enter)
  2. Wait for it to finish the process
  3. Purge (Enter)
  4. Save the drawing

After saving the drawing, you'll find your file size reduced.

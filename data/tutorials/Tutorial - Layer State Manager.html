What is the status of a layer means?
  1. Layer is on
  2. Layer is off
  3. Layer is locked
  4. Layer is unlocked
  5. Layer is freezed
  6. Layer is un-freezed

Following are the steps to use Layer State Manager:



Step 01: First all layers of you file are to be on, unlocked and un-freezed. To achieve that, enter following commands;

layon (enter)
laythw (enter)

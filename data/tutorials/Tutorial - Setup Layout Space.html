1. Decide Scale

This is a Model space and it contains two different plans.


So the next thing you need to do is, Go to Layout space in which you want to import these plans.

2. MVIEW Command

There is a command "MVIEW" or "MV" to add a viewport into Layout space.
 
That viewport will reflect the data in Model Space into Layout space.

Follow these steps to do it,

2.1. Enter command "MVIEW" or "MV"

Command "MVIEW" or "MV" will create a viewport in Layout space which will reflect the content of Model Space.
Note that this command won't run in Model Space.


2.2. Specify Corners in viewport

As explained in the image above, AutoCAD is asking to specify the First Point/Corner from where you want to draw the viewport.
After clicking the first point, you'll be asked to specify Opposite Point/Corner where you want to end the viewport.


It is similar as drafting a Rectangle.

Note that one viewport can have only one Scale. Here there are two plans to set in two different scales.
That is why you'll have to add another viewport for the another plan.

After Creating a viewport, Next thing you'll need to do is:

3. Set Scale

3.1. Double-click inside viewport

Which will allow you to access these options in image below:



Clicking dropdown menu for Scale will open the same,
Now there are multiple options of scales, from which you'll have to select the required one.
So select 1:50 for the one Viewport and 1:75 for the another.


Now if you can not find the scale you want in the given options, this is how to create it: 

3.2. Create Scale

Go down and select Custom button in the list:


Which will open the following menu:


Select Add button to add a new scale to list, which will open following menu:


Let's say that you want to add 1:75 scale,
So you'll have to add an appropriate Name,

Set Paper Units to 1 and Drawing units to 75.

Click OK to finish the process and Scale 1:75 is now available in the list.

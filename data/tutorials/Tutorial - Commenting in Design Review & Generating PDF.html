Introduction

Design Review is the free software by Autodesk. Which, is created to review (as the title of software says it self) documents, add comments, and add stamps.
One can import many formats into Design review, Some of them related to architecture are as mentioned: 
  • AutoCAD File
  • PDF File
  • Image File
Before we start, Make sure Design Review & A PDF generator (Check what is PDF generator in Chapter 2 if you have no idea about it) is installed in your PC.


1. Commenting in Design Review

Let me explain first about how to,
     1.1.     Export an AutoCAD file to Design Review
     1.2.     Import PDF/Images to Design Review

1.1. Export an AutoCAD file to Design Review

  • Make sure you have your layout set up (Refer Tutorial - Loading CTB & Plotting in our MSA Standard folder if you have no idea about how to set a layout for printing)
  • Enter keyboard shortcut Ctrl+P from Layout tab in AutoCAD
  • Change the Plotter name to DWFx as shown in the image below
  • Choose location where you want to save the file
  • Double click on that file, it'll open it in Design Review

1.2. Import PDF/Image file to Design Review

  • Open Design Review software
  • Click Design Review Icon
  • Click Open
  • If you don't want to follow this long process, you can directly use keyboard shortcut: Ctrl+O
  • Select the file that you want to import and it's done

1.4. Controlling Automatic Palette Launching  

You'll find Palettes like MarkUps, MarkUp Properties, Text Data, Grid Data, Layers etc on your screen.
So when you select any object in the document, one of the related palette will pop up, and most probably you shall not need to use any of it's functionality.

The only Palette you shall need is MarkUp Properties,
Where data of your MarkUps will be stored.

Find out how to turn off the other Palettes here:

  • Open Options from the Design Review Icon drop down


  • Now uncheck all the options except Markup Properties


  • Click OK and you're done with that.

2. Generating PDF from Design Review

You'll need a PDF Generator to generate the PDF from Design Review,
Confused about how to find it?

Follow these steps:
  • Click Ctrl+P (Which will open the Print Toolbox
  • Find Name and open the drop down menu
  • Find relevant plotters (doPDF / AdobePDF or any you have installed in your PC)

If you can't find it, Contact Administrator.

If it's already installed, Follow these steps:

  • Click Ctrl+P (Which will open the Print Toolbox
  • Select the relevant plotter Name to generate PDF
  • Select Paper Size (I would prefer Letter/A4)
  • Select Orientation as per your requirement


And now comes the interesting Part: If you want your comments To Standout or To get merged with data

  • You'll find Print Color section in the Print Toolbox
  • Sheets*: To keep the sheet as it is, Go to Sheets dropdown and select Color, Use Grayscale to convert sheet into Grayscale style
  • Markups*: To keep the sheet as it is, Go to Sheets dropdown and select Color, Use Grayscale to convert sheet into Grayscale style

You should keep Sheet style to Grayscale & Markup style to Color to get your comments To Standout,
You can use other combinations of styles as per your requirement.


  • Click OK when you're done with setting the page
  • Choose location where you want to save the PDF

And one last step:

  • Give it some time. When the process is complete, you'll find the PDF in the location you chose.

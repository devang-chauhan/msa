1. Loading CTB

If you're facing the problem to load the CTB or don't know how to load it,
Here's a stepwise tutorial about how to do it:

  1. Enter the command 'OP' and Options toolbox will be opened.
  2. Go to Files

When you're done with that,

  1. Go to Printer Support Path
  2. Then go to Plot Style Table Search path
  3. Select the path
  4. Click Browse


*Path shown here is correct, But may not be the same in your AutoCAD file.

After clicking Browse, A new toolbox will be opened, in which you'll have to search for the CTB folder.
Here is the Location where you'll find the CTB folder:

Working Project (\\192.168.1.100) ► MSA TEMPLETS ► ACAD_Customization ► CTB



You'll find that all the CTBs have been loaded in your Plot Setup Manager after this process:




2. Plotting

Here is the tutorial about how to Plot the Layout:

Step 01: Page Setup Manager

Right Click on the Layout and Select Page Setup Manager.



Page setup toolbox will be opened after clicking Modify
In what, Certain settings should be set:

  1. Name: DWG to PDF.pc3
  2. Paper size: Whichever the paper size of the layout is. (ISO A4/A3/A2/A1/A0)
  3. Select Window in What to Plot
  4. Click on the Window and Select the Border of your layout to Plot.
  5. Plot style table: Select the CTB. (Eg. If the layout size is A0, A0_Detail.ctb must be used)
  6. Keep quality Normal
  7. Set drawing orientation (Eg. If your layout orientation is Portrait, Select Portrait)
  8. Only keep these options always Activated:
          Center the plot
          Fit to paper
          Scale lineweights
          Plot object lineweights
          Plot with plot styles
          Plot paperspace last


You can click on Preview to check if the Page setup is done properly or not.
Once you are done with that, Hit OK.

Now, You won't have to set up your Plot Settings everytime you enter Ctrl + P,
And the Settings which were applied in Page Setup Manager will remain as Default Settings. Overall, It'll save the time.

Step 02: Plot

01. Plot to PDF

  1. Press Ctrl + P
  2. Click OK

Note that you have already set your page up, so you don't have to set it up everytime when you want to plot.

02. Print

  1. Press Ctrl + P
  2. Name: Select the printer that you want to print from
  3. Click on Properties
  4. Select Custom Properties from listed items
  5. Click on Custom Properties
  6. Make sure if Factory Defaults is selected
  7. Select the paper size you want
  8. Select the paper source. It maybe varied in every printer. (Eg. My printer holds A4 size papers in Tray 1, So the Tray 1 is selected)
  9. Hit OK
  10. Hit OK in Plot toolbox to give a Print.

These commands, LAYISO & LAYUNISO are used to Isolate or Unisolate layers.

1. LayISO

This command is used to Isolate layer(s),
You should not work on particular layer with all the other layers activated,
Isolating Layer(s) will make the job easy for you.

So now, Freezing all the unnecessary layers will take more time,

To Isolate objects,

  • Enter the command LAYISO or
  • Use the Layer Isolate icon as shown in image below:


After entering the Command or Clicking the icon, You'll be asked to select the layers to Isolate:


After selecting the layers that you want to work on, Hit Enter/Space. And those layers will be isolated then:


That is how Isolating the layers and then work will make your job easier.

2. LayUNISO

This command is used to Unisolate layer(s),
After finishing the work in isolated layers, using the Layer Unisolate will bring all the layers back just right they were before.

3. LayISO Settings

If you find the other layers (which you don't want to isolate) get locked, You'll need to change the LayISO settings.
Here's the stepwise introduction about how to do it:

  1. Enter command LAYISO orÂ Layer Isolate icon
  2. Enter S (Settings)
  3. Enter O (Off)



When you're done with setting it up, All the layers that you don't want to isolate will go 'Off' instead of being 'Locked' the next time you use LayISO.

Using Design Review for Commenting Drawings and Documents

Last updated on : 23-12-2017

Design Review is the free software by Autodesk. Which, is created to review documents, add comments, and add stamps. Design review takes a lot of document formats. Some of the most used are .dwg , .pdf, .jpg. You can import your AuotCAD drawings, PDF documents, and images in Design Review to comment.

Before we start, Make sure Design Review & A PDF creator is installed on your system. PDF Architect or doPDF are two creators used here at MSA. Check in the start menu if any of them are installed on your system.

How to export from AutoCAD
  • Make sure you have your layout properly set up for plotting. To know more about this, please check this tutorial. If you have more than one sheets in this file and you wish to send all the sheets for review, you are expected to send a combined DWFX file. You can check this tutorial to know how to generate a document with multiple sheets.
  • Enter keyboard shortcut Ctrl+P from sheet (paper space) in AutoCAD.
  • Change the Plotter name to DWFx as shown in the image below
  • Choose location where you want to save the file
  • Double click on that file. It'll open in Design Review
Import PDF or Image to Design Review
  • Open Design Review
  • Click the Design Review icon
  • Click open
  • Select the file you want to import
Look for Markup & Measure tab. This you'll find next to the Home tab
Add callouts as shown in the image above
Add text to callouts as shown in the second image
You can change the type of leader to a markup
Draw shapes, lines, clouds, and add text
Measure something in the drawing. In some cases, this dimension will not be as accurate as AutoCAD. Use this for a quick check only
Format the text, lines, and other drawing objects in the Formatting palette
Add a stamp to your document after have reviewed the document
You'll find palettes pop up when you select an object. If you want to change the palettes that open, you can change that in the options. Click on the Design Review icon and click on options
Typically, you only want masrkup properties, so uncheck the rest
Click OK and you have your preferences saved. The next section shows how to export the commented sheet as a PDF
